Content Rules

Max length
~65 characters visible (1 line on desktop, truncate on mobile). Two data points separated by middot.
Precision gate
~95% confidence minimum before displaying any AI-extracted or derived content. If below threshold, fall back to structured TMP data.
Explainability
When content is personalized (not default), prefix with signal context: "Based on your search," "Matches your filters," or inline match language.
Cross-surface parity
SRP card content must not contradict what the renter sees on BDP/HDP or in Voyager responses. Same data source, same language.
Differentiators only
Show attributes NOT already covered by active filters. If renter filtered for "pet friendly," don't repeat "Pet friendly" in the dynamic row — show pet fees/deposit instead.
Fair housing
All personalized content must pass fair housing review. No demographic targeting. Personalization based on explicit user actions only.
